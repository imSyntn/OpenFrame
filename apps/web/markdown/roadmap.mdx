# Roadmap

- User Authentication and Profiles

- Image Optimization and BlurHash Placeholders
- Advanced Search (Tags, Users, and Photos)
- Paginated Results
- Image Downloading
- SEO Metadata and Schema.org Integration
- Markdown Documentation System
- Fully Responsive UI
